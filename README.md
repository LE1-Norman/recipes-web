# recipes-web
Java web application for Russian food recipes

This website presents various recipes of Russian cuisine. 

The site supports the implementation of certain functions in asynchronous mode, it implements work with cookies and user authentication with restricted access rights.
It is possible to view a list of recipes, search for recipes according to various criteria, add recipes and leave reviews. The website is also available in English and Russian.
When implementing the work, Java, JSP, servlets, HTML, JS, Apache Tomcat were used.
